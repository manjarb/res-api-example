# Rest API Example

# Technologies
1. Node JS 10.x
2. PostgreSQL 9.6.x
2. Express JS 4.x
3. TypeORM 0.2.x
4. TypeScript 2.x

## Steps to run
    1. Create a user table  by executing the schema from  /schema/user.sql.
    2. Modify the /src/config.ts. (Replace it with the a real db connection details)
    3. Install all the dependenies by running `npm i`
    4. npm run dev
